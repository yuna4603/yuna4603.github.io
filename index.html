    • 创建好的仓库点击setting进入仓库的设置页面,将分支设置为master，并且找到blog地址，用于之后配置到_config.yml文件中

    • cd进入Myblog目录中，编辑_config.yml文件

    • 你需要将repository换成你自己的gihub的用户名

添加ssh key到Github避免每次都需要输入用户名和密码

如果本地没有配置github账号，之后每次提交可能都需要输入用户名和密码，使用起来很不方便，因此需要设置一下。如果你的电脑之前已经配置过ssh key，那么这一步可以跳过。

  1. 检查本地电脑是否有ssh key, 进入~/.ssh目录下查看是否存在文件id_rsa.pub和id_dsa.pub，如果已经存在，可直接进行步骤2，将ssh key添加到github上，如果不存在，则输入下面命令生成ssh key，其中的邮箱地址换成你自己的github邮箱

  2. 将~/.ssh目录下的id_rsa.pub文件中的内容复制到github的ssh key中。
    进入Github –> Settings –> SSH keys –> add SSH key添加

创建第一篇文章

blog搭建好了，之后就可以愉快的编写文章了

  1. 创建新文章
    • 进入blog项目目录下，执行hexo new My New Post,其中My New Post就是你的文章名称，执行之后，会source/_post目录下生成一个My-New-Post.md的文件，这时就可以打开这个文件愉快的写你的第一个博客了。
  2. 编辑后发布
    • 编辑完文章之后，保存，然后回到blog目录下，执行命令
      hexo g 生成静态页面
      hexo s 进行本地预览
      hexo d 部署发布到远程

到此为止属于你自己的blog就算搭建成功啦~下面再介绍一些自定义设置让你的blog更丰富。
